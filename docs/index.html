📋 Task Manager API

A Spring Boot REST API for managing tasks

Overview

The Task Manager API provides a simple CRUD interface for managing tasks. It uses an in-memory H2 database, so data resets on each restart.

This page is hosted on GitHub Pages — deployed automatically via GitHub Actions whenever files in docs/ change.

API Endpoints

Method Endpoint Description
GET /api/tasks List all tasks
GET /api/tasks/{id} Get a specific task
GET /api/tasks/status/{status} Filter by status
POST /api/tasks Create a new task
PUT /api/tasks/{id} Update a task
DELETE /api/tasks/{id} Delete a task

Example Request

curl -X POST http://localhost:8080/api/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Learn GitHub Features",
    "description": "Explore Actions, Pages, Wiki, and more",
    "status": "TODO"
  }'

Response

{
  "id": 1,
  "title": "Learn GitHub Features",
  "description": "Explore Actions, Pages, Wiki, and more",
  "status": "TODO",
  "createdAt": "2026-03-30T10:00:00",
  "updatedAt": "2026-03-30T10:00:00"
}

Task Model

Field Type Description
idLongAuto-generated ID
titleStringRequired, max 200 chars
descriptionStringOptional, max 1000 chars
statusEnumTODO, IN_PROGRESS, or DONE
createdAtDateTimeAuto-set on creation
updatedAtDateTimeAuto-updated on changes
